# App Mute

音が出ているアプリごとに個別のミュートショートカットを設定できるツールです。
SteelSeries Sonar（仮想オーディオデバイス）にも対応しています。

---

## 必要なもの

- Windows 10 / 11
- Python 3.10 以上

---

## インストール手順

### 1. Pythonをインストールする

https://www.python.org/downloads/

インストール時に **「Add Python to PATH」にチェックを入れてください。**

### 2. このリポジトリをダウンロードする

右上の緑の「Code」ボタン →「Download ZIP」→ 解凍する

### 3. 必要なライブラリをインストールする

解凍したフォルダをエクスプローラーで開き、アドレスバーに `cmd` と入力してEnterを押す。

表示されたコマンドプロンプトに以下を貼り付けて実行：

```
pip install pycaw psutil keyboard requests comtypes
```

### 4. 起動する

```
python mute_app.py
```

---

## 使い方

1. 起動すると、音が出ているアプリが自動で一覧に表示されます
2. 設定したいアプリの入力欄をクリック
3. 登録したいキーを押す（例: Ctrl+Alt+M）
4. 自動で保存されます
5. 以降そのキーを押すたびにミュート／解除が切り替わります

---

## アンインストール

フォルダごと削除してください。レジストリは使用していません。
